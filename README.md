# Phone Check

ეს პროექტი ახლა ორ რეჟიმს შეიცავს:

- `index.html` - ბრაუზერული დიაგნოსტიკა პირდაპირ ტელეფონზე
- `desktop.html` - Android ტელეფონის მართვა კომპიუტერიდან `ADB`-ით

რას ამოწმებს:

- ეკრანის ფერები და dead pixel-ის ვიზუალური ტესტი
- touch ეკრანის ზონები
- კამერა
- მიკროფონი
- სპიკერი
- ვიბრაცია
- motion/orientation სენსორები
- ქსელის მდგომარეობა
- GPS/ლოკაცია
- ბატარეის სტატუსი, როცა ბრაუზერი ამის საშუალებას იძლევა

## გაშვება

პროექტის საქაღალდეში ყველაზე მარტივი გაშვებაა:

```powershell
python serve.py
```

ალტერნატიულად შეგიძლია გამოიყენო სტანდარტული სერვერიც:

```powershell
python -m http.server 8000
```

შემდეგ ბრაუზერში გახსენი:

```text
http://localhost:8000
```

PC control panel მისამართია:

```text
http://localhost:8000/desktop.html
```

თუ ტელეფონიდანაც გინდა გახსნა, კომპიუტერის IP ნახე:

```powershell
ipconfig
```

და ტელეფონზე გახსენი მსგავსი მისამართი:

```text
http://192.168.1.10:8000
```

ორივე მოწყობილობა ერთ Wi-Fi ქსელში უნდა იყოს.

მნიშვნელოვანი:

- `http://localhost:8000` კარგია მაშინ, როცა ამავე კომპიუტერზე ტესტავ.
- თუ ტელეფონიდან შეხვალ `http://192.168.x.x:8000` მისამართზე, ეკრანი, touch, ვიბრაცია და ნაწილი სენსორების იმუშავებს, მაგრამ კამერა, მიკროფონი და GPS ბევრ ბრაუზერში დაიბლოკება.
- ამ სამი ტესტისთვის უკეთესია აპი განათავსო HTTPS ჰოსტინგზე, მაგალითად GitHub Pages, Netlify ან Vercel.

## Android კომპიუტერიდან სამართავად

თუ გინდა კომპიუტერიდან მართო Android ტელეფონი:

1. ტელეფონში ჩართე `Developer options`
2. ჩართე `USB debugging`
3. USB-ით შეაერთე კომპიუტერს
4. ტელეფონზე დაადასტურე `Allow USB debugging`
5. კომპიუტერზე დააყენე `Android Platform Tools`, რომ `adb.exe` გქონდეს

შემდეგ `desktop.html` გვერდიდან შეძლებ:

- მოწყობილობის ნახვას
- ბატარეის, storage-ის და Android ვერსიის ამოღებას
- კამერის გახსნას
- Settings/Wi-Fi პარამეტრების გახსნას
- screenshot-ის აღებას
- `scrcpy`-ის გაშვებას, თუ ისიც დაყენებული გაქვს
- reboot/home/volume ბრძანებებს

თუ მერე კაბელის გარეშე გინდა:

- პირველად USB-ით დააპეარე
- შემდეგ შეგიძლია გამოიყენო `adb tcpip 5555`
- და დაუკავშირდე `adb connect ტელეფონის_IP:5555`

## შენიშვნა

ბრაუზერს არ აქვს სრული წვდომა ტელეფონის ყველა hardware ნაწილზე, ამიტომ ღრმა დიაგნოსტიკა
როგორიცაა CPU throttling, modem health, internal storage SMART ან battery cycle count,
აქ პირდაპირ არ იკითხება.

ასევე:

- `desktop.html` ნაწილი გათვლილია Android-ზე.
- iPhone-ზე ასეთი დონის მართვა კომპიუტერიდან ბევრად უფრო შეზღუდულია.
